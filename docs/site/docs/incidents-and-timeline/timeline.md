# Timeline

The timeline is the raw chronological event feed across all nodes and sources.

## What You Can Do

- Filter by node.
- Filter by source.
- Search full text across entry content and service names.
- Review notes attached to individual entries.

## Entry Shape

Entries typically include:

- ULID ID
- Timestamp
- Node name
- Source
- Service
- Event type
- Content
- Metadata

## Search

Blackbox uses SQLite FTS5 for full-text search and highlighting behavior.
