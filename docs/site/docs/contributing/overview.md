# Contributing Overview

Contributions are welcome across code, docs, tests, source integrations, and
operational polish.

## High-Value Contribution Areas

- Fixes for incident correlation and event normalization
- New or improved data sources
- Documentation improvements
- Better deployment and troubleshooting guidance
- UI and admin usability improvements

## Before You Start

- Read [Project Structure](./project-structure.md)
- Set up a local dev environment
- Identify whether your change affects the server, agent, web UI, docs, or more
  than one area

## If You Add Behavior

Please also update:

- Tests
- User-facing docs
- Contributor docs when the workflow changes
